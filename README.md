# OptiGrade Pro – OMR Grading System

## Description
OptiGrade Pro is a web-based OMR (Optical Mark Recognition) grading system designed for small-scale educational institutions and tuition centers.

## Features
- Generate custom OMR sheets
- Upload and evaluate scanned responses
- Automated grading system
- Web-based (no installation required)

## Technologies Used
- Python
- HTML, CSS, JavaScript
- SQLite

## Use Case
Helps teachers and institutions quickly evaluate objective exams without manual checking.

## Future Improvements
- Add user authentication
- Improve accuracy of image detection
- Cloud deployment
